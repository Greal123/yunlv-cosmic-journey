# I Built a Cozy Game ✨ *Vital Pulse* with Claude Code — Looking for Playtesters!

Hey everyone! I've put a lot of heart into this little world and can't wait to share it with you all. Using Claude Code, I've designed **Yun Lü (蕴率)**, a cozy companion game free from the grind of combat and leveling up. It's all about peaceful relaxation, quiet reflection, and gentle self-companionship — a temporary safe haven for whenever you need it. 🌌

## 🌿 Game Storyline
Journey from the vastness of the cosmos, drift through the cycle of four seasons, wander into the quiet rhythm of daily human life, encounter the wonders of the animal kingdom, and finally return to the boundless universe. Each scene holds a unique mood, ready to immerse you no matter how you're feeling.

## ✨ Core Gameplay (Super Easy to Pick Up)
1. **Scene Navigation**  
   On every screen, choose `go on` to move forward or `stay here` to linger and zone out — entirely at your own pace.
2. **Light Dot Creation**  
   Type text while staying on a page to generate unique floating light dots, and save as many as you like. Click a dot to change its color, scroll to resize it, or double-click to make it fade away. Once all dots disappear, you'll automatically advance to the next scene.
3. **Custom Music & Save System**  
   Upload local audio files via the music icon in the top-left corner to set your own background music. Save anytime from the top-right corner, and load your progress from the homepage — no need to worry about losing your session.

## 💫 Cozy Little Details
Designed from the start to be a quiet companion for daydreaming, thinking, memorizing vocabulary, jotting down inspirations, or acting as your personal time capsule. Turn your current feelings and random thoughts into glowing dots, and watch them float gently across the screen for a wonderful sense of comfort and security.

This is the initial version, and there's still plenty I want to refine and improve! I'd truly appreciate it if you'd give it a try and share your feedback. I'll keep polishing this little world. Come explore your own peaceful corner in *Vital Pulse*!

## 🚀 How to Run

### Prerequisites
- Node.js (v14 or higher)

### Installation & Running
```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

Then open your browser and visit `http://localhost:3000`

## 🛠️ Tech Stack
- **Three.js** - 3D graphics and cosmic scenes
- **Canvas 2D** - Procedural animations and effects
- **Vite** - Fast development and build tool
- **Vanilla JavaScript** - Pure JS, no framework dependencies
